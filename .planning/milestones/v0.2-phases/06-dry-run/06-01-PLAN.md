3. Print stacked annotations — `(dry-run)` first, then `(filtered by --only "...")` if applicable. Do NOT print `(cache disabled)` even if `--no-cache` is passed (silently ignored per user decision). Print blank line after annotations.

4. Iterate tests and check cache status. Use `test.case` (NOT `test.name`) for cache lookup. Resolve baseUrl as `test.baseUrl ?? config.baseUrl ?? ""`. Compute max name length for padding.
   ```typescript
   const maxNameLen = Math.max(...config.tests.map(t => t.name.length));
   let cachedCount = 0;

   for (let i = 0; i < config.tests.length; i++) {
     const test = config.tests[i];
     const baseUrl = test.baseUrl ?? config.baseUrl ?? "";
     const entry = await cacheManager.load(test.case, baseUrl);
     const source = entry ? "cache" : "ai";
     if (entry) cachedCount++;

     const paddedName = test.name.padEnd(maxNameLen);
     console.log(`  ${i + 1}. ${paddedName}  (${source})`);
   }
   ```

