    StepRecorder:
    - record() appends a step with toolName and toolInput
    - getSteps() returns a copy of all recorded steps
    - clear() empties the steps array
    - wrapTools() returns a new tools object where each tool's execute calls the original and records on success
    - wrapTools() does NOT record when the original tool execution throws an error
    - getSteps() returns empty array initially

    StepReplayer:
    - replay() executes all steps in sequence and returns { success: true } when all succeed
    - replay() stops on first failure and returns { success: false, failedStep: N, error: message }
    - replay() with empty steps array returns { success: true }
    - replay() calls the ToolExecutor with correct toolName and toolInput for each step
  </behavior>
